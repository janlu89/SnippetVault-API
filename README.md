# SnippetVault API

A personal code snippet library built with .NET 10 and Clean Architecture.
The frontend repository is maintained separately.

## Tech Stack
- .NET 10 / C# 14 / ASP.NET Core Web API
- Entity Framework Core 10 with SQLite (dev) and SQL Server (production)
- JWT authentication
- Serilog for structured logging
- xUnit + Moq for unit testing
- OpenAPI with Scalar UI

> Full documentation coming in v1.0 release.
